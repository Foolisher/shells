## 是不是线程越多效率越高

**答：**不是，多线程技术利用了现代多核CPU并发处理的优势，对一个任务可以并发地执行，以提高执行效率。但是并不是线程数量越高执行效率越高，因为线程超过CPU数量或线程进入阻塞状态时CPU将会对无法就需的线程调度到阻塞状态，而此刻会有线程上下文切换的开销，若这样的切换很频繁带来的代价也挺大的；
