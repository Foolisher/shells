![](/assets/tmf-扩展点.png)流程

> 一个产品方案一个流程组合，一个流程是有若干节点组成，把各个节点串联起来就成了一个产品业务流程

## 节点

> 一个流程中有若干节点，各个节点承载了这个节点的一些动作

## 拓展

> 一个产品有标准流程节点，但是不同业务方有自己实现逻辑，需要做到业务隔离，业务识别，不同业务方的链路走不同扩展点覆盖逻辑
